This is an example blog post. You can delete this file and create your own posts in the `_posts/` directory.

## How to create a new post

1. Create a new file in `_posts/` with the format `YYYY-MM-DD-title.md`
2. Add front matter at the top (like in this file)
3. Write your content in Markdown below the front matter

## Front matter options

```yaml
---
layout: post
title: Your Post Title
subtitle: Optional subtitle
tags: [tag1, tag2]
comments: true          # Enable comments (requires config)
cover-img: /path/to/img # Large header image
thumbnail-img: /path/to/img # Small thumbnail
share-img: /path/to/img # Image for social sharing
---
```
